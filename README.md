# EDA_Practice

This repository contains multiple exploratory data analysis (EDA) projects for practice and learning purposes. Each folder contains a self-contained EDA project with its own dataset, analysis notebook, and supporting files.

## Projects

### 1. Book_Sales
- **Description:** EDA on a book sales dataset, exploring trends in publishing year, genres, sales, ratings, and more.
- **Main Files:**
  - `Books_Data_Clean.csv`: Cleaned dataset of book sales and metadata.
  - `eda-on-book-sales.ipynb`: Jupyter notebook with data cleaning, analysis, and visualizations.
  - `README.md`: Project-specific documentation.

### 2. Boston_Housing_Dataset
- **Description:** EDA on the Boston Housing dataset, including data exploration and possibly image data for classification tasks.
- **Main Files:**
  - `archive/boston.csv`: Main dataset for housing prices.
  - `archive/index.ipynb`: Jupyter notebook for analysis.
  - `archive/dataset/`: Contains image data for cats and dogs (for classification practice).

### 3. Startup_Funding
- **Description:** EDA on startup funding data, with both notebook and app-based analysis.
- **Main Files:**
  - `startup_funding.csv`: Dataset of startup funding rounds.
  - `index.ipynb`: Jupyter notebook for EDA.
  - `app.py`, `streamlit_doc.py`: Streamlit apps for interactive data exploration.

## How to Use
1. Navigate to the project folder you are interested in.
2. Open the main notebook (`.ipynb`) file in Jupyter or VS Code.
3. Ensure you have the required Python libraries installed (see each project's README for details).
4. Run the notebook cells to reproduce the analysis and visualizations.

## Requirements
- Python 3.x
- Common data science libraries: pandas, numpy, matplotlib, seaborn, etc.
- Jupyter Notebook or VS Code (recommended for interactive exploration)
